HTML

  • HTML stands for Hyper Text Markup Language
  • HTML is the code used to structure a webpage and its content.
  • HTML consist of a series of elements.
  • Elements tell the browser how to display the content.
  • The title element displays the title on the pages tab on the browser.
  • The head element contains information about the webpage.
  • The body element represents the visible contect shown to the user.
  • The ul and ol elements are used to seperate data into a list (ul as bullet points, ol as a numbered list).

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS tells the browser how the elemnts in HTML should be displayed.
  • Three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class attribute is used to define a group of HTMML elements in order to apply styling to those elemnts separately.
  • A link element is used to connect an outisde file to the HTML code.
  • href is the attribute that specifies the location of the external source (URL).
  • rel attribute specifies the relationship between the current document and the linked document resource.

Git

  • A Repository a.k.a repo is a collection of soruce code.
  • git add is a command usde to add a file that is in the working directory to the staging area.
  • git commit is a command used to add all files that are staged to the local repository.
  • git push is a command used to add all committed files in the local repo to the remote repo.
  • git pull is a command used to get files from the remote repository directly into the working directory
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.

  • String is a sequence of text.
  • An Aray is a structure that allows you to store multiple values ina single reference.
  • Conditionals are code structures used to test if an expression returns true or not e.g. if...else statements.
  • A function is a block of reusable code written to perform a specific task.
