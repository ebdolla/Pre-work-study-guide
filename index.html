HTML

  • HTML stands for Hyper Text Markup Language
  • HTML is the code used to structure a webpage and its content.
  • HTML consist of a series of elements
  • Elements tell the browser how to display the content
  • The title element displays the title on the pages tab on the browser
  • The head element contains information about the webpage
  • The body element represents the visible contect shown to the user
  • The ul and ol elements are used to seperate data into a list (ul as bullet points, ol as a numbered list)

CSS

  • CSS stands for Cascading Style Sheets
  • CSS tells the browser how the elemnts in HTML shoudl be displayed
